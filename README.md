#### Git Commands - #### <em>`Utilities`</em>
`Global`

<em>Config user && email</em>

``` 
git config --global user.name <username>
```

``` 
git config --global user.email <email>
```

<em>Show config</em>

``` 
git config --global -e
```

<em>Change default branch</em>

``` 
git config --global init.defaultBranch <name>
```

<em>Disable crlf</em>

``` 
git config --global core.autocrlf true
```

`Branches`

<em>List branches</em>

``` 
git branch -a
```


<em>Rename branch</em>

``` 
git branch -m <oldName> <newName>
```

`Repository`

<em>Start File Tracking</em>

```
git init
```

<em>Ways to upload files</em>

``` 
git add . 
```

``` 
git add *.extension
```

``` 
git add folderName/
```

<em>Commit</em>

``` 
git commit -m "Here goes the message"
```

<em>Add && Commit together - Only files that have tracking</em>
``` 
git commit -am "Here goes the message"
```
